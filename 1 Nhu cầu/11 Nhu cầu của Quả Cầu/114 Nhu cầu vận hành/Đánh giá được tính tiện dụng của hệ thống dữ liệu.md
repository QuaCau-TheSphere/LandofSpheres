
Phục vụ cho Thành quả cần có:
```dataview
list where contains(thành-quả-cần-có, [[]]) or contains(thành-quả-hỗ-trợ, [[]]) or contains(ý-tưởng-tốt-hơn, [[]]) 
```

```dataview
list where contains(tương-ứng-với-thành-quả-cần-có,[[]])
```
