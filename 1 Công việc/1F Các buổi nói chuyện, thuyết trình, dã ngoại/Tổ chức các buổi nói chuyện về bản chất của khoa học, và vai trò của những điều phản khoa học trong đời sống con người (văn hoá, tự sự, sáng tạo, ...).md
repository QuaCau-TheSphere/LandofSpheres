Phục vụ cho mục tiêu:
```dataview
list where contains(hoạt-động, [[]])
```
Nằm trong công việc lớn hơn:
```dataview
list where contains(công-việc-thành-phần, [[]])
```
Hướng tới đối tượng:
```dataview
list where contains(row["Giải pháp"],[[]])
```
Phục vụ cho nhu cầu:
```dataview
list where contains(row["Cách đáp ứng nhu cầu này"],[[]])
```
