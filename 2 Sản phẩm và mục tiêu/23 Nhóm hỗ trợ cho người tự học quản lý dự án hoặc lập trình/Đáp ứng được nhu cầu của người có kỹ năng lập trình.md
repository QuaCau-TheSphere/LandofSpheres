Phục vụ cho mục tiêu:
```dataview
list where contains(mục-tiêu, [[]]) or contains(yếu-tố-hỗ-trợ, [[]]) or contains(ý-tưởng-tốt-hơn, [[]]) 
```

```dataview
list where contains(tương-ứng-với-mục-tiêu,[[]])
```
